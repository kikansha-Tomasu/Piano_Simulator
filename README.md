# Piano（iPad Web ピアノ）

ブラウザで動作するシンプルなWebピアノアプリです。iPad向けに最適化されていますが、モダンブラウザであれば PC でも動作します。

## 機能

- **音域**: C3〜B5（3オクターブ、36音）
- **タッチ対応**: マルチタッチによる和音演奏が可能
- **録音・再生**: 演奏を録音してそのまま再生できる
- **外部依存なし**: HTML/CSS/JavaScript のみで動作（ライブラリ不要）

## 使い方

## キーボード操作

- 白鍵・黒鍵をタップ（またはクリック）すると音が鳴ります
- 横スクロールで全音域（C3〜B5）にアクセスできます
- 複数の鍵盤を同時に押さえて和音演奏が可能です

## 録音・再生

| ボタン | 操作 |
|--------|------|
| ● 録音 | 録音を開始する（赤く点滅） |
| ■ 停止 | 録音または再生を停止する |
| ▶ 再生 | 録音した演奏を再生する |
| ✕ 消去 | 録音データを削除する |

## 技術仕様

- **音声合成**: Web Audio API（デュアルオシレーター、0.3%デチューン）
- **エンベロープ**: アタック 8ms / ディケイ 300ms / サステイン 0.20 / リリース 500ms
- **動作環境**: Web Audio API 対応ブラウザ（Safari、Chrome、Edge など）
